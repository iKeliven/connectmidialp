O desafio

Cada grupo deverá criar uma solução visual completa para um projeto fictício, passando pelas principais etapas de uma campanha multimídia.

Design Vídeo Interfaces Multimídia

01. Identidade Visual

Criação de logo, paleta de cores, tipografia, conceito e aplicações visuais.

02. Vídeo

Produção de um vídeo de apresentação com roteiro, edição e linguagem criativa.

03. Landing Page

Desenvolvimento da interface de uma página de divulgação no Figma.

04. Pitch Final

Apresentação final dos projetos desenvolvidos durante o evento.
